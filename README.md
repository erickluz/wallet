# Endpoints

**/saldo**

Saldo da conta

**/meusdados/**

Dados da conta e do cliente e logado.

**/transferencia/**

Dados da conta de destino da transferencia, e o valor a ser transferido.

**/extrato/**

Historico de transferencias da conta

**/pagamento/**

Dados do boleto a efetuar o pagamento.
